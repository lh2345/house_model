# house_model
这个是一个房屋模型的测试仓库

## 100 平一层住宅 3D 设计图

本项目提供一个可交互的 3D 房屋设计网页，使用真实比例规划约 10m x 10m 的单层住宅。建筑面积约 100㎡，其中套内功能区约 91㎡，其余为墙体与结构占用；空间包含客餐厨、主卧、次卧/书房、卫生间、玄关、储物和南向露台。

### 使用方式

直接用浏览器打开 `index.html` 即可查看。若浏览器限制本地 ES Module（模块）加载，可以在仓库目录启动一个静态服务：

```bash
python3 -m http.server 8080
```

然后访问 `http://localhost:8080`。

页面通过 CDN 加载 Three.js（3D 图形库），预览时需要联网。

### 交互方式

- 左键拖拽：旋转视角
- 滚轮或触控板：放大/缩小
- 右键拖拽：平移模型
- 页面按钮：切换屋顶、墙体透明度和房间标注
